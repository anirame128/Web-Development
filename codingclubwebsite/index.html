BHS Coding Club

 

Have you ever wanted to code but did not have time or resources to learn it? Then Coding Club is the club for you! You will go through the basics of many programming languages such as: Java, JavaScript and HTML/CSS. Using this knowledge, you will build small and fun projects like applications and games; and later on in the year, you can compete in Hackathons and coding competitions.

Meet on September 13th from 3-4 p.m. in C109. See you guys there!
